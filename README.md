# Saudi-Dialect-Analysis-Dataset-NLP-Pipeline

Worked on building a Saudi Arabic dialect dataset collected from multiple sources to represent five major Saudi dialects. The data was gathered from open resources provided by SDAIA (SADA صدى dataset) and from publicly available YouTube audio content.

The purpose of this dataset is to support Arabic AI applications, particularly in areas such as speech recognition and dialect analysis. The dataset is designed to reflect real-world spoken Saudi Arabic and focuses exclusively on Saudi dialects, making it suitable for region-specific language technologies.

One identified limitation of the dataset is a bias toward the Najdi dialect, due to the higher availability of data sources compared to other Saudi dialects.

SADA link: https://www.kaggle.com/datasets/sdaiancai/sada2022
